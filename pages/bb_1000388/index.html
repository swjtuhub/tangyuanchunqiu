唐院春秋备份

罗河教授上书（二）—义正词严 状告郭“贼”

| | | 关于郭维城挑动群众斗群众迫害教师
造成三条人命的一些具体材料
（罗河教授上书二）
A、材料清单

1、1966年6月13日563期“新唐院”的部份照片（附件1）
2、郭维城1966年6月8日讲话摘要全文（见附件2）
3、563期“新唐院”其他各文的目录和要点（见附件3）

B、事件发生的地点和过程
1、 1964年铁道部决定把唐山铁道学院迁到四川蛾眉山旁一个山沟里。这个校址是由部长吕正操亲目选定的．现在发现除地处偏僻农村不宜作重点高等学院的基地外还存在污染问题，人到那里不久就白血球大量下降，暗中影响健康。此外由于地质条件复杂，建设进展很慢，到I966年6月主要房屋建成很少。只初步形成一片纷乱昀建筑工地。先后三人自杀事件就在1966年6月到9月间在这片工地上发生的。
2、 据听过郭维城讲话的人说，除院刊的讲话摘要外，他还说：“吕部长要我管一管学校的运动” ；“我派了两个打手崔德顺，戈平来学校”。崔、戈二人即以副院长名义参加学院工作，按郭维城讲话的精神领导运动。
3、 在1966年6月8日郭维城来工地讲话的同时，以吕正操部长为首的铁道部领导派出办公厅主任刘伯涛到唐山市对唐山铁道学院传达部令，决定把全院师生员工调往四川峨眉，在山沟里闹革命。住房不够就搭临时席棚。经过报告，讨论，准备行装，分批出发。最后一批以教授、副教授为主的人员在刘伯涛的押送下于1966年6月18日到达峨眉。此时峨眉建校工地的战斗空气经郭维城几次讲话挑动巳高涨到一触即发之势。
4、 563期“新唐院”转载的两篇人民日报社论当然是林彪四人帮的阵阵妖风；但仍然停留在“专家”，“学者”，“权威”等空洞名词上。就是按照四人帮的标准，这种帽子谁该戴谁不该戴也不具体明确。但郭维城则对四人帮的谬论领会的特别深透。他从未进过唐院的大门，更没有和哪位“权威”有过一面之缘，而他竟能信口开河在群众大会上指名道姓地乱扣帽子，这只能说明他紧跟四人帮，仇恨知识分子，乘机打击迫害、摧残教育，破坏正常秩序，为四人帮一伙帮了大忙。
5、 563期“新唐院”很明显是为贯彻郭维城讲话精神组织稿件对知识分子进行围攻迫害的檄文。其中两篇社论是纲，而讲话摘要通过点出几位教授的名把全体教授作为打击迫害的对象。其他五篇文章全是配合郭维城的讲话写的。三篇里都有“郭维城同志报告后”字样。有的喊“不需要你们这些教授权威，要和你们彻底清算”，有的喊“开展积极斗争，……不获全胜，决不收兵”等。很明显这期院刊是根据郭维城的意旨由打手们主持出笼的。
6、郭维城在讲话里说：“半年来了四次唐院，讲三次话”，“三通战鼓揭开了激战的序幕，形势很好，昨天为止已经贴出了4211份大字报”，“我点火欢迎你们放火，烧谁都可以”。形势正如“新唐院短评所说的，“势如暴风骤雨，迅猛异常”。一些没有斗争经验的知识分子陷入火海，惊恐万分。 52年怀着满腔热情为建设祖国由美国返回大陆的电机系副系主任姚皙明（女）副教授再也不能控制住自己，就在6月中旬由附近一个小桥头上跳下来跌死，成为“抗拒运动，自绝于人民的反革命”。
7、按照郭维城的说法（见其讲话摘要）学院党委是没有问题的，学生是革命的，唯独教授是顽固的洋奴，是找匾的奴才。他一连在大会上点了五位教授的名，把他们和早巳在反右中上吊自杀的胡思杜（胡适的儿子）相提并论．真是恶毒已极。
郭维城仇恨知识分子迫害教授比四人帮更凶狠．更露骨；他亲自跳出来点名点火，号召“要发动广大群众，大闹唐院”。“在一周之内写出两千张大字报，给北边送”，，“以南压北”，要比四人帮更上一层楼。就在这种“势如暴风骤雨迅猛异常”的气氛中，一批以教授、副教授为主体的人员在铁道部办公厅主任刘伯涛的押送下于66年6月18日到达蛾眉建校工地。
当他们乘的卡车—进建校工地，这批蒙在鼓里的教授就被分开，拉到预定地点进行殴打批斗：但除自己外．谁也不知道别人的遭遇。到1978年12月我才听范治纶教授说他亲眼看到杨耀乾教授连眼镜都被打破了。
那次我被批斗的问题是“找匾”；但我事先完全不知道有这样一个莫须有的罪名。被冒雨斗了两小时，什么具体罪行也没有交代出来；群众无奈，只得领我去看大字报。一看大字报，我才明白，原来是郭维城在背后给我戴的帽子，怎么也否认不了。在随后的岁月里，头顶郭记“找匾”的大帽子，日子是不好过的。如果没有坚强的信心认为总有一天会再见天日，早该跟姚皙明她们走了。
9、郭维城“要发动广大群众，大闹唐院”号令果然战果累累，很快就无中生有地揭发出日本特务张鸿逵（二级教授，立即被捕，关押多年，到中日建交才无事释放），“曹潘轴心”，唐院的三家村”，唐院的“吴晗”等，凡是北边有的．南边都有．这场场迅猛异常的暴风骤雨，内容丰富多采有“集体罚跪”，“戴高帽挂牌游行”，“坐飞机”，“拳打脚踢”，“两手背绑吊在空中几上几下”等各种残酷体罚，把没有斗争经验的知识分子吓得不知所措，成为过街老鼠无地自容。除姚皙明（女）副教授已于6月中旬眺山自杀外，又有解放前夕为了效忠祖国由英国返回大陆的机械系主任史家宜教授投河目杀。
66年8月下旬，史家宜教授先在厕所里上吊被发现抢救复苏。不久他又被迫跪在路旁接受批斗。他哀嚎说“我不仅精神上受不了，肉体上也受不了”。但他的哀嚎对郭维城的号令“运动的来势就是要猛”不起任何作用。事后他就乘机投河自杀，成为第二个“自绝于人民的反革命”。
10、郭维城在讲话里还以“干打垒”昀权威自居，说什么“干打垒，要土石并举，以土为主，多种多样，标新立异。唐院要创出一个全国最好的干打垒，要树样板。”
后来实践证明，在多雨和易风化的山坡上，干打垒根本不行；但在郭维城大力吹捧的初期，谁也不敢怀疑它的效用。后来干打垒房屋倒塌了，负责房屋建筑的姚富州讲师就受到很多责难，也曾被批斗过。他感到样板难树罪责难逃，乃于66年9月初上吊自杀，成为第三个“自绝于人民的反革命”。很明显，他的自杀是和郭维城的话分不开的。
在66年6月18日以后的一段时间里，铁道部办公厅主任刘伯涛以工作组组长的身份住在现场按上级领导意图会同学院负责人指导运动，对所发生的惨案也不能说完全没有责任。
11、郭维城身为外单位的干部，即或受吕正操的私人委托代管唐院的运动，也只应按中央政策办事，而决不应当亲自跳出来，对完全不了解的人和事信口胡说把斗争矛头引向一般知识分子，以致把俩位教授和—位讲师逼得自杀。一清二楚的事实表明姚、史、姚三人自杀是郭维城挑动群众斗群众的结果，与四人帮完全无关．在后来平反昭雪时，把这笔账记在四人帮头上而让主角郭维城滑过去，那是很不公平的。这是要提请中央领导主持公道的问题之一。
12、由于院刊“新唐院”的报道和流传全国各地都知道唐山铁道学院有一块“军阀时代，一位教育部长给学校写过—块匾”，有一个“找匾”的“奴才罗河”，并有一个“老顽固”的“洋奴罗忠忱”，把在国内外有一定名望的唐山交通大学描绘成漆黑一团．事实果真如此吗？
先说那块匾。 1916年唐山交通大学的前身，唐山路矿学堂以它毕业生王节尧的毕业论文在全国各大学教学成绩展览会上取得第一名光荣称号，并由当时政府发给—块匾作为纪念。这是一次真刀真枪、没有任何投机取巧的成绩比赛。在任何历史时期的公开比赛中，冠军都是光荣的。几十年来唐院的师生员工和校友都引以自豪，以此鞭策自己，努力工作和学习，力争不辱前人取得更好的成绩以报答全国人民。这块起着促进鼓励作用的纪念品无论从什么角度看都是富有进步作用的。
但在四人帮反动思想的毒害下，在他们的眼里，一切好的都是坏的：不独匾是可耻的，就连享有这种荣誉的学府唐山交大也不应当继续存在。于是唐山交大就于1971年被迫全部迁到四川峨眉县一个偏僻的山沟里，改名换姓，称为西南交通大学。但由于选址不慎。这个西南交大存在两个根本问题，第一是偏僻农村和外界联系困难．不符合国家重点大学校址的规定，影响教学质量的提高。
第二个问题是山区水土气候不正，可能还有特殊污染问题，导致一些流行性疾病如腮腺炎，传染性肝炎，疟疾，流感等常年发生，教职工体质普遍下降，白血球明显减少；1 977年对108名教师进行抽查，白血球平均数为5779土173相当于国家统一标准的最低限．其中最低的只有2000左右（见校党委1979年3月13日报告）这是一种无法补救的致命伤。由于以上两种原因，教师们情绪低落，身心不宁，外地生望而却步，不肯前往就学。这就使这所本来名牌的人才工厂由于处理不当，产品质量下降不可能和外边兄弟大学并驾齐驱，同步前进。从事教育工作的人都知道办好一所大学不是容易事。—所历史悠久名闻中外经历过辛亥革命，北洋军阀，国民党反动统治和八年抗日战争等巨大政治动荡仍安然存在，并在解放后为人民立了新功的学府竟会无故被消减掉，是国家—个重大损失。
四化要人才，教育是根本。原先唐山交大的教师们怀念过去的业绩，都希望在原来地区恢复唐山交通大学．现在国内外的校友也有同样要求。一些外国友人也以素有盛名的学府不得继续存在为之惋惜。打倒四人帮后，冤案得到平反；错案得到纠正。一些停办了的院校多在原地恢复。把我国这所历史悠久、成绩卓著的工科大学唐山交大恢复起来使它能在原有基础上继续壮大，提高为四化大业作出贡献是符合多快好省的原则的。这是提请中央考虑的问题之二。
13、郭维城在群众大会上公开造谣，骂罗忠忱是老顽固是洋奴，对我们进行恶毒的攻击，若果如此，他早就会在反右运动中被打成右派了。
不管是古今中外，这都是破坏别人名誉的犯罪行为。由于“新唐院”在全国流行，罗忠忱及其家属亲友所受损害是严重的。罗忠忱教授为人正直不阿，忠于自己的岗位工作，严格要求．一丝不苟．在半个世纪里几次负责学校领导工作，但从未放下手中的粉笔。他是我国早期一位不务虚名的教育家。他的模范作用在门徒中有深远的影响。以毕业论文为唐院评得第一荣誉的就是他的学生王节尧。国民党的反动头子陈立夫任伪教育部长时曾亲自到校托人敦请罗忠忱教授在教师中带头集体加入国民党，但被他严辞谢绝。仅此一端即可见罗忠忱教授的为人。解放后他已年过古稀，仍坚守教学岗位到批准退休为止。他的学生中不乏知名之士；一般毕业生都在岗位上以一当十地发挥骨干作用。他们对这位老师都非常尊敬。地方党政领导对他也很重视．唯独郭维城奉行四人帮的反动哲学不独在大会上公开骂他，还通过宣传工具“新唐院”在全国败坏他的名誉，使这位当时已86岁的老教授卧病在床还不免臂加黑箍蒙受“黑五类”的耻辱。这桩冤案极待按党的原则平反昭雪，这是请求中央考虑的问题之三。
14、由当年总统徐世昌亲笔书写的“竢实扬华”四字匾由1916年起一直到解放后一段时期悬挂在唐山交大的学生饭厅里，前后几十年成为新老同学交谈的话题．高班同学向我介绍过；我也向后来的同学同事介绍过；这本来是一件不足为奇的普通事．而郭维城为了当上极左的“革命派”竟不择手段硬把什么“找匾”的帽子从背后扣在别人头上，以便进行打击迫害；并利用宣传工具在广大范围内败坏别人的名誊。这是十足的犯罪行为。
尽管受了很多冲击，现在我仍然健在，继续为人民做些力所能的工作，并不计较这点个人得失。今天所以揭发郭维城一伙的问题，主要因为他当时跳得很高，在广大群众面前做过有声有色的表演。并且通过“新唐院”在全国广为宣传．对他的民愤是极大的，而他至今还没有按党的传统，在适当场合公开检查自己的问题，更谈不到取得人民的谅解．这类犯有诽谤罪，挑动群众斗群众造成三条人命的人，不管有多大能力，过去有多大功劳，现在混在上层领导班子里，就是十目所视，十手所指的污点。事关中央领导的威信，不容缄默，谨就所知，如实反映，幸垂察焉！此上中共中央华主席，邓副主席，陈副主席。

【附录2】：唐山铁道学院院刊“新唐院在其1966. 6. 13第563期里刊载了一篇与唐院没有任何关系的郭维城的报告摘要，全文如下 ：
高举毛泽东思想伟大红旗
为把栽院建成一所非常无产阶级化，非常革命化的新唐院而
斗 争
铁路建设工地指挥部党委付书记郭维城同志在我院师生员工大会上的报告摘要。
（本刊讯）六月八日下午五时，我院师生员工集会于工厂区，由郭维城同志给师生员工做了关于当前文化大革命的报告，广大师生员工情绪缴昂，充满了浓烈的革命气氛，决心一定要高举毛泽东思想伟大红旗．把文化大革命进行到底，不获全胜决不收兵！

郭维城同志的报告摘要如下：
郭维城同志亲切地向大家说：半年来了四次唐院，讲三次话。这次来晚了一点，同学们认为党委有些迟疑，但这不是主要方面，主要是成果问题。有十几位同学提出，唐院问题是否跟北大一样，有的提出搞“革”小组。唐院与北大都是老资格了，有相同的地方但唐院党委与原北大党委问题性质不同。唐院是在铁道部党委和四川省委领导下的，北大党委是在前北京市委资产阶级代理人领导下的。陆平与前北京市委不但在思想上而且在组织上都是“三家村”的人。这个问题要明确。至于党委领导成员个别有问题那是另外一回事。开始有些乱，但也有好处，三通战鼓揭开了激战的序幕形势很好，昨天为止已经贴了4211份大字报。
同志们，我这次来不是来灭火的，是来点火的。运动的来势就是要猛，大烧一场，烧着谁算谁。烧是烧掉坏思想，我讲错了，你们也可以贴大字报。我点火欢迎你们放火。烧谁都可以。
最近报纸发表了几篇社论和一篇宣传要点，是对无产阶级文化大革命最高最全面的指示和总结。大家要认真学习。
接着郭维城同志讲述了我院的一些具体情况及搞什么？怎么搞？
唐院有七十年的历史，就是要翻这七十年的老账。解放前是洋大学。解放后十几年唐院走的那条路？什么思想统治唐院？要算清，算清过去为了将来。要按照毛泽东思想指引的道路前进。有破有立，必须先破后立，彻底的破，破中有立，大破大立。工学院与文法学院有所不同，文法全属于上层建筑。工科有技术问题，但也有上层建筑意识形态问题，要认识客观，改造客观同时也要改造主观。
当前这场文化大革命是一场触及人们灵魂的大革命，谁都有阶级立场、政活思想问题，所以谁都不例外。
唐院情况不熟悉。把同志们反映的几个问题提供参考，做为星星之火，希望成为燎原之势。唐院与北大争年岁，说北大你是1898年，我是1896年，不但与北大争，还与美国的康乃尔排辈，说什么是“东方的康乃尔”。军阀时代一伪教育部长给学校写过一块匾，找不到了，后来罗河找匾，党员院长也下令找匾。真是大奇事。这是资本主义思想，这是奴才思想。
罗忠忱（C.C.罗）是个老顽固，解放后对我们一句好话也没有说过，还对我们进行恶毒的攻击。罗忠忱是个洋奴，上课都用外文讲课，可是，唐院把他做为偶象崇拜。他生活并不困难，却在困难时期送他200元钱，两条毯子。当时多少老工人和贫下中农家庭出身的学生生活困难，谁这样关心过？这是立场问题。
给伍镜湖做寿，顾院长去讲了话。顾院长母亲死了，她是个地主婆．有人出钱给买花圈。五年级学生反映，五年没见过副院长顾宜孙的面。胡适的儿子胡思杜以前在学院教马列主义，真是怪事到处有，唯有唐院多。
同学们，把唐院的丑事大大地揭揭，揭他们的丑，叫他们痛。
郭维城同志讲到关于我院突出政治问题时说，毛泽东思想是否挂帅了，政治是否挂帅了？毛主席说政治是统帅，是灵魂是一切工作的生命线．政治必须统帅一切，一切必须为政治服务。绝不能把这个关系颠倒过来，唐院是否颠倒了，没有颠倒也歪了。没有突出政治。突出了也不够用。用毛泽东思想武装全体人员，实现人的思想革命化，有了精神成果必然会有物质成果，什么人间奇迹都可以创造出来。
总支，支部领导权还没有完全拿过来，政治工作没有摆在适当位置上，唐院不问政治，轻视政治相当严重。
唐院与资产阶级讲和平共处，讲平等，这个不行！被剥削阶级同剥削阶级不能和平共处，不能讲平等，无产阶级对资产阶级的斗争，是你死我活的斗争。旧社会什么时候与我们讲和平共处，讲过平等呢？他们不是任意屠杀和压迫我们吗？
放毒也不行。毛主席说“凡是错误的思想；凡是毒草，凡是牛鬼蛇神，都应该进行批判，决不能让它钔自由泛滥。”
据说你们学院中有人说学毛主席著作是“简单化”，“庸俗化”，说什么“政治听党的，业务听教授的”，“政治落实到业务”。都错了！党是领导一切的，突出政治首先是落实到人的思想革命化。还有人说什么“文化继承论”，“抽象的思维教育”。他反对毛泽东思想进课堂，孙训芳说什么：“不是方向，不足为训”。弄得大家都不愿做政治工作，把一个学习毛主席著作的积极分子搞得灰溜溜的。
学生一进门，大标语上：“欢迎你，未来的工程师”。这不是变相的高价收买吗？还说什么“分数是学生的命根子，考试是教师的法宝”。毛主席著作才是命根子，才是法宝，贫下中农子弟要入校门很难。据说一个贫农子弟考试不及格非勒令退学不可。
同志们提出要对院务委员会夺权，刀把子要夺过来。
大学十六、七年太长，见不到工农，越读越蠢。要搞点实际滚一身泥巴。现在你们参加建校，已经看不出那是工人那是学生了。这就好。我们的方向是缩小、消灭三大差别。
关于文化大革命问题，要先搞大是大非，搞思想政治问题。
郭维城同志谈到建校问题时说，文化大革命，思想革命化，也将大大促进建校。以前讲过一次，主要是反对“洋”，突出“土”。这关系到缩小三大差别问题。大部份人通了，—部份还不通，有人反对，有人抵抗，学生赞成，贫下中农说对头，方向要坚持，不能动摇．要低标准，低造价，高速度高质量，就地取材，因地制宜。
干打垒，要土石并举，以土为主，多种多样，标新立异。唐院要创出一个全国最好的干打垒，要树样板。
给石油学院的挑战书送去了吗？没有送去，六月底要送去，说了就办。
要向峨眉河要料，到年底还要搞四万平方米。建校问题也有两种思想，两条道路的斗争。以前，就是想往大城市搬，我说这里好得很。家属插队是方向，不能动摇：双职工多，也不能少于50%的家属去插队。
总之，唐院与北大性质虽然不同，但问题严重。要发动广大群众，大闹唐院。要南北并重互相呼应，以南压北，以求全胜。一周之内能不能写两千张大字报给北边送去?要根据毛主席的指示，横扫一切牛鬼蛇神。
同志们！百分之七十的工农出身的子女，和其他阶级出身的表现好的大部分同学都是要革命的，你们是文化大革命的主力军，也是革命的接班人。革命的教职员大部分是好的会同你们一起闹革命的，不会落后的，要来个大决战。
至少停课一周。一定打出一个以毛泽东思想挂帅的，非常无产阶级化，非常革命化的新唐院来，不获全胜，决不收兵！
